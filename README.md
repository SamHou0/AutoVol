# AutoVol

## What is AutoVol

AutoVol can change your system Volume automatically when you change your foreground app.

## Feature

- Create a list of apps to change volume
- Set the volume of each app

## Environment

- Visual Studio 2022
- dotnet 6.0 with winforms
